Giano Della Corsa Automation Project

Giano Della Corsa is an AI-driven automation system that connects n8n workflows with the TikTok API to automatically generate and publish high-quality automotive videos.

This integration is part of the GRANDPRIXBRASIL brand — a project dedicated to performance, technology, and speed. All videos, descriptions, and hashtags are generated using intelligent agents and scheduled through n8n automations.

For more information, visit n8n.io or contact the project maintainer.
